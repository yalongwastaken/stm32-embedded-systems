# STM32 Embedded Systems

A growing collection of STM32 projects exploring core embedded systems concepts using the STM32 HAL framework.

## Structure

Projects live under `src/`, organized by topic.

```
src/
└── ...           # More to come
```

Each project contains its own README with details and setup instructions.

## Tools

- **Board:** NUCLEO-F446RE (STM32F446RE)
- **Framework:** STM32 HAL
- **IDE:** STM32CubeIDE
